# Video Saver - Личная коллекция видео

## 📱 О проекте
Веб-приложение для сохранения и организации видео из TikTok, YouTube Shorts, VK и локальных файлов.

## 🚀 Функции
- Сохранение ссылок на видео
- Загрузка видео с компьютера/телефона
- Категории и теги
- Просмотры и статистика
- Адаптивный дизайн (ПК + мобильная версия)

## 📁 Структура
- `index.html` - основная версия для ПК
- `m.index.html` - мобильная версия
- `styles.css` / `m.styles.css` - стили
- `script.js` / `m.script.js` - логика
- `assets/` - иконки и изображения

## 🔧 Установка
1. Скачайте все файлы
2. Откройте `index.html` в браузере
3. Нажмите "Добавить видео" и вставьте ссылку или загрузите файл
